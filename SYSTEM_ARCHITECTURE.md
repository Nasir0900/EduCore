# EduCore ERP System Architecture

## Project Vision

EduCore ERP is a modern University Management System developed using ASP.NET Core MVC, Entity Framework Core, SQL Server, and Bootstrap.

The goal is to create a scalable, modular, and production-ready ERP that can manage the complete academic and administrative workflow of universities and colleges.
