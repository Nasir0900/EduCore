## 📖 Overview

EduCore ERP is a comprehensive University Management System designed to automate and simplify academic and administrative operations.

The system is being developed with scalability, maintainability, and modern software engineering principles in mind. It follows a modular architecture, allowing institutions to manage academic, financial, library, and human resource operations through a single platform.

This project is being built as a real-world portfolio application and follows professional development practices, including Git version control, documentation, modular design, and continuous improvement.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Nasir0900/EduCore.git
```

Navigate to the project

```bash
cd EduCore
```

Restore packages

```bash
dotnet restore
```

Apply migrations

```bash
Update-Database
```

Run the application

```bash
dotnet run
```
